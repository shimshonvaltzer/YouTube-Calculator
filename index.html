📋 צעדי ביצוע מהירים Guide
1

צור מאגר (Repository)

כנס ל-GitHub, צור מאגר חדש בשם "my-channel" והגדר אותו כ-Public.

2

הוסף קובץ index.html

הדבק את הקוד שקיבלת לתוך קובץ חדש בשם index.html ושמור (Commit).

3

הפעל את GitHub Pages

בהגדרות המאגר (Settings), תחת Pages, בחר בענף Main ולחץ Save.
